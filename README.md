# Huda Portfolio 🌐

A responsive personal portfolio website built using HTML, CSS, and JavaScript.

## 🔗 Live Demo
[Click here to view](https://2025-AsrulHuda.github.io/Huda-Portfolio)

## 📌 Features
- Home section with profile photo and bio
- About section with skills and education
- Projects showcase with images and descriptions
- Certificates section
- Social links (LinkedIn, GitHub)
- Dark/Light mode toggle
- Smooth animations

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript (Vanilla)
